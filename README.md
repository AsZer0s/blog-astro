# ObsiBlog

![Node.js >= 20](https://img.shields.io/badge/node.js-%3E%3D20-brightgreen) 
![pnpm >= 9](https://img.shields.io/badge/pnpm-%3E%3D9-blue) 

一个基于 [Astro](https://astro.build) 构建的静态博客系统。

> **📌 重要说明**：本项目是基于 [saicaca/fuwari](https://github.com/saicaca/fuwari) 模板的**魔改版本**，感谢原作者的优秀工作！

## ✨ 特性

- [x] 基于 [Astro](https://astro.build) 和 [Tailwind CSS](https://tailwindcss.com) 构建
- [x] 流畅的动画和页面过渡效果
- [x] 响应式设计，完美适配移动端
- [x] 基于 [Pagefind](https://pagefind.app/) 的全文搜索功能
- [x] Markdown 扩展语法支持
- [x] 文章目录（TOC）
- [x] RSS 订阅支持
- [x] 置顶文章功能
- [x] 多平台分析统计集成（Microsoft Clarity、Google Analytics、Cloudflare Web Analytics、TianJi）
- [x] 自动提交 sitemap 到百度搜索

## 🧩 Markdown 扩展语法

除了 Astro 默认支持的 [GitHub Flavored Markdown](https://github.github.com/gfm/)，还包含以下扩展功能：

- **Admonitions（提示框）**：支持 note、tip、important、warning、caution 等类型
- **GitHub 仓库卡片**：使用 `::github` 指令展示仓库信息
- **增强的代码块**：基于 Expressive Code，支持代码高亮、行号、代码折叠等功能
- **数学公式**：支持 LaTeX 数学公式渲染
